# 🎉 酒桌游戏 - 微信小程序

一款让聚会嗨起来的酒桌游戏小程序，支持多种卡牌类型和自定义卡组。

![License](https://img.shields.io/badge/license-MIT-orange)

## 📖 项目简介

**酒桌游戏**是一款运行在微信小程序平台上的派对游戏工具。它提供了丰富的卡牌库，包含真心话、大冒险、互动游戏等多种类型，让聚会不再冷场。

---

## ✨ 功能特性

### 🎴 卡牌系统
- **4 大基础卡组**：真心话、大冒险、互动、地狱
- **10 大主题卡组**：家庭聚会、同学聚会、户外露营、情侣约会、同事团建、朋友小聚、生日派对、沙雕整蛊、深夜畅聊、经典老歌
- **自定义卡组**：支持创建、管理、编辑个人卡组
- **卡组导入**：可以从官方卡组快速导入内容

### 🎮 核心玩法
- 点击抽卡，快速获取随机任务
- 支持多卡组同时启用
- Canvas 粒子动画背景，效果炫酷
- 卡牌来源标识，清晰分辨官方/自定义

### 📱 页面结构
| 页面 | 功能 |
|------|------|
| `pages/index/index` | 抽卡首页，粒子动画 + 抽卡 |
| `pages/main/main` | 卡组管理，我的卡组列表 |
| `pages/new-deck/new-deck` | 新建/编辑卡组 |
| `pages/deck-detail/deck-detail` | 卡组详情预览 |

## 🚀 快速开始

### 环境要求
- 微信开发者工具（最新版）
- 微信小程序账号

### 运行步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/porall/wineGame.git
   cd wineGame
   ```

2. **导入微信开发者工具**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择项目目录
   - 填入 AppID：`wx8d121f2590124b8e`

3. **配置云开发环境**
   - 在微信开发者工具中开通云开发
   - 创建环境（记录环境 ID）
   - 修改 `app.js` 中的环境 ID

4. **预览/发布**
   - 点击"编译"预览效果
   - 上传代码审核后发布
